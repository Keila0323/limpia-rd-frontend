# Limpia RD — Frontend

A single-page frontend for the **Limpia-RD marketplace API** — connecting property owners and Airbnb hosts in the Dominican Republic with verified cleaners and service providers.

**Live site:** [keila0323.github.io/limpia-rd-frontend](https://keila0323.github.io/limpia-rd-frontend/)  
**Backend API:** [limpia-rd.onrender.com](https://limpia-rd.onrender.com)  
**API Docs:** [limpia-rd.onrender.com/docs](https://limpia-rd.onrender.com/docs)  
**Backend repo:** [github.com/Keila0323/Limpia-RD](https://github.com/Keila0323/Limpia-RD)

---

## Features

- Browse verified cleaners filtered by city, service type, language, and rating
- Submit hospitality cleaning service requests with date/time scheduling
- AI-powered matching — surfaces the top 3 cleaners for each request based on location, service type, and rating
- Live API integration with graceful fallback if the backend is cold-starting
- Fully bilingual (English / Spanish)
- Mobile-responsive, accessible design

## Tech Stack

**Frontend:** HTML, CSS, JavaScript (vanilla)  
**Backend:** FastAPI · Python · SQLAlchemy · SQLite · Stripe  
**Hosting:** GitHub Pages (frontend) · Render (backend)
